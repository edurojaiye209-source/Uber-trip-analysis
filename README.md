# 🚖 Uber Trip Data Analysis

## 📌 Project Overview
This project focuses on analyzing Uber trip data to uncover patterns in travel behavior, trip frequency, and customer usage.

## 🎯 Objectives
- Check how long people travel with uber ?
- What hour do most people take uber to their destination?
- Check the purpose of the trips?
- Which Day has the highest number of trips?
- What are the trips in the month?
- The starting point of trips. Where do people start bording their trip from the most?

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Key Insights
- Friday recorded the highest number of trips
- Business trips were more frequent than personal trips
- Certain locations had higher pickup demand
- Trip frequency varied across different days

## 📂 Dataset
Uber trip dataset (sample dataset for analysis)
